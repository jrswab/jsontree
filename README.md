# jsontree - JSON Tree

Go package for printing JSON as a Tree in the terminal.

**Example:**

```bash
/
└── 0
    ├── Architecture: amd64
    ├── Author:
    ├── Comment:
    ├── Config
    │   ├── AttachStderr: false
    │   ├── AttachStdin: false
...
```
